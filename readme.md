# 项目名称
OpenGL+bullet physics自由落体仿真

## 简介

简要说明项目的功能和目标，例如：
> 这是一个基于 OpenGL 和 Bullet Physics 的三维物理仿真项目，用于模拟真实物理场景中的物体运动和碰撞。

---

## 功能特性

列出项目的主要功能点：
- [x] 使用 OpenGL 渲染三维场景
- [x] 集成 Bullet Physics 引擎
- [x] 支持刚体模拟和碰撞检测
- [x] 平台动态地面和自由物体交互
- [x] WS键控制物体移动
- [x] QE键控制物体转向
---
