# 项目名称
OpenGL+bullet physics自由落体仿真

## 简介

基于OpenGG和Bullet Physics的自由落体仿真项目，实现了三维场景的渲染、刚体模拟和碰撞检测、平台动态地面和自由物体交互。

## 功能特性

键盘UIO可以旋转移动相机，
